# Detection frame assets

Each file is one original PNG image embedded in the anonymous presentation; no screenshot crops, generated imagery, or pre-composed strips are used.

- `real/`: head and wrist detection frames from `video/ral_video.pptx`, organized as task / clean-or-random setting / camera view. The filename preserves its original media index (`ral-image-N`).
- `sim/`: RoboTwin clean/random detection frames for hammer, stand, basket, phone, and plate. The filename preserves its original media index (`source-N`).
