# Anonymous MoQ project page

This repository is display-only. It contains the anonymous project page, paper figures, matched video posters, and the selected clean/disturbed experiment videos.

It intentionally excludes paper sources, author notes, bibliography files, local build outputs, presentation sources, and unrelated recordings.

## Publishing

GitHub Pages is deployed by `.github/workflows/pages.yml`. Select **GitHub Actions** as the Pages source once in repository settings.

For double-blind review, share only the resulting link through [Anonymous GitHub](https://anonymous.4open.science/anonymize), not the underlying GitHub repository URL.

The empty PDF, Code, and arXiv buttons can be activated after review by editing `script.js`.
